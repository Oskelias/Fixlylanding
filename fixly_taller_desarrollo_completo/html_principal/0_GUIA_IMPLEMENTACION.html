Archivos del Sistema

login.html

Sistema de login con validación de códigos de activación y credenciales de clientes pagos.

Funciones:
  • • Validación de códigos FXT-XXXX
  • • Control de 15 días de prueba
  • • Login de clientes pagos
  • • Redirección al sistema principal

registro_nuevo.html

Formulario de registro actualizado que genera códigos automáticamente.

Funciones:
  • • Generación automática de códigos
  • • Simulación de envío de emails
  • • Validación de formularios
  • • Redirección a login

backend_admin.html

Panel administrativo para gestionar códigos, usuarios y emails.

Funciones:
  • • Dashboard de códigos activos
  • • Gestión de usuarios
  • • Control de expiraciones
  • • Estadísticas de conversión

mercadopago.html

Configuración completa de MercadoPago para suscripciones.

Funciones:
  • • Botón de suscripción $19.900
  • • Webhook para confirmación
  • • Generación de credenciales
  • • Email post-pago
